## TripTune을 소개합니다 ! 

## ✈️ 프로젝트 소개
Triptune 프로젝트는 피그마 등 협업 툴로 일정 계획을 세우는 사용자 경험을 웹으로 구현하고자 시작했습니다. 사용자들이 여행지 정보와 일정을 쉽게 추가하고, 공유할 수 있는 서비스로, 여행 계획에 특화된 기능을 중심으로 개발되었습니다.
